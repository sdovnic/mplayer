MPlayer for Windows

MPlayer packaged with ready to run scripts and binding to file extensions. Includes 3D, Audio extract, Playlists, Karaoke Subtitles, AniDB file check and more.

Download 64bit Version

Instructions

Extract the archive to your desired location and run the installation script. To use the AniDB file check edit shell/avdump2.ini.contrib and rename it to avdump2.ini.

If your download did not start, try to redownload.
